1. Introduction                           *vim-tmux-aisend-intro*

This plugin formats selected code with file path, line numbers, and syntax
type, then either copies it to the clipboard or sends it to a tmux pane.

Typical use case: selecting code in Vim and pasting it directly into a Claude
Code / AI session running in an adjacent tmux pane.

The send target can be auto-selected (largest non-current pane) or manually
specified via interactive prompts with completion, similar to tslime.vim.

==============================================================================
2. Usage                                       *vim-tmux-aisend-usage*

Copy for AI (~ay):
    Visually select lines, press <leader>ay. The formatted text is copied to
    the system clipboard.

Send to AI (~at):
    Visually select lines (or press in normal mode for the current line) and
    press <leader>at. The formatted text is sent to the target tmux pane.

    Target resolution order:
        1. g:ai_tmux_target_pane — manual override
        2. g:ai_send_target — set via |:AITarget|
        3. Auto-select — largest non-current pane
        4. Interactive prompt — if auto-select returns no target

    So on first use you can just press <leader>at directly — if no target
    is configured, it will prompt you to pick session:window.pane.

Set target (~as):
    Press <leader>as to interactively select session, window, and pane with
    tab completion.  The target is stored in |g:ai_send_target|.

==============================================================================
3. Mappings                                 *vim-tmux-aisend-mappings*

<Plug> mappings (always available):

    <Plug>CopyForAI           Visual: copy selection to clipboard
    <Plug>SendSelectionToAI   Visual: send selection to tmux/clipboard
    <Plug>SendLineToAI        Normal: send current line to tmux/clipboard
    <Plug>SetAITarget         Normal: interactive target selection

Default key mappings (disable with |g:ai_tmux_no_mappings|):

    <leader>ay                Visual: copy selection
    <leader>at                Visual/Normal: send to tmux
    <leader>as                Normal: set tmux target interactively

==============================================================================
4. Commands                             *vim-tmux-aisend-commands*

    :[range]CopyForAI    Copy range to clipboard with context
    :[range]SendForAI    Send range to tmux (or clipboard fallback)
    :AITarget            Interactively set tmux session:window.pane
    :AIClearTarget       Clear manual target, revert to auto-select

Examples:
    :'<,'>CopyForAI      Copy visual selection
    :%SendForAI          Send entire buffer
    :10,20SendForAI      Send lines 10 through 20

==============================================================================
5. Manual Target                           *vim-tmux-aisend-target*

Like tslime.vim, you can manually specify the exact tmux pane to send to.

Setting the target:

    :AITarget

Prompts for session, window, and pane with tab completion:
  - Session: lists all tmux sessions
  - Window: lists all windows in the chosen session
  - Pane:   lists all panes in the chosen session:window

Once set, every <leader>at / :SendForAI will send to that pane.

    :AIClearTarget

Clears the manual target and reverts to auto-select.

The current target is stored in |g:ai_send_target| as a dictionary:
    {'session': 'mysession', 'window': '1', 'pane': '2'}

You can also set it directly in your vimrc:
    let g:ai_send_target = {'session': '0', 'window': '0', 'pane': '1'}

Priority for target resolution:
  1. |g:ai_tmux_target_pane| if set (raw pane_id, e.g. '%1')
  2. |g:ai_send_target| if set (session:window.pane)
  3. Auto-select largest non-current pane (default)

6. Configuration                           *vim-tmux-aisend-config*

                                *g:ai_tmux_target_pane*
g:ai_tmux_target_pane
    Force a specific tmux pane ID (e.g. '%1').  When empty (the default),
    falls through to g:ai_send_target or auto-select.

                                *g:ai_tmux_interactive*
g:ai_tmux_interactive
    Set to 'pane' or 'window' to prompt interactively every time you send,
    instead of using a preset or auto-select.
      'pane'   → prompts for pane only (session+window auto-selected)
      'window' → prompts for window, then pane
      0        → preset / auto-select mode (default)

                                *g:ai_tmux_fallback_clipboard*
g:ai_tmux_fallback_clipboard
    When vim is not inside tmux, fall back to copying to clipboard.
    Default: 1 (enabled).

                                *g:ai_tmux_autoset_pane*
g:ai_tmux_autoset_pane
    When selecting a pane via |:AITarget|, skip the pane prompt and
    automatically select the tallest (most content) pane.
    Default: 0 (disabled).

                                *g:ai_tmux_always_current_session*
g:ai_tmux_always_current_session
    When selecting a session via |:AITarget|, show only the current
    session instead of listing all sessions.  Default: 1 (enabled).

                                *g:ai_tmux_always_current_window*
g:ai_tmux_always_current_window
    When selecting a window via |:AITarget|, show only the current
    window.  Default: 1 (enabled).

                                *g:ai_tmux_no_mappings*
g:ai_tmux_no_mappings
    Set to 1 to prevent default <leader> mappings (ay/at/as).
    You can then map <Plug> yourself.

                                *g:ai_send_target*
g:ai_send_target
    Dictionary storing manual target: {'session': 's', 'window': 'w',
    'pane': 'p'}.  Set via |:AITarget| or directly.

                                *g:ai_tmux_use_absolute_path*
g:ai_tmux_use_absolute_path
    Whether the file path in the output is absolute or relative to the
    working directory.  0 = relative (default), 1 = absolute.

                                *g:ai_tmux_raw_content*
g:ai_tmux_raw_content
    Output format.  0 (default) = send file path, line numbers, and code
    fence.  1 = send only the raw selected text, no metadata.

Example:
    let g:ai_tmux_autoset_pane = 1
    let g:ai_tmux_no_mappings = 1
    nmap <leader>st <Plug>SetAITarget
    vmap <leader>ss <Plug>SendSelectionToAI
